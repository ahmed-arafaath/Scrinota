Scrinota

Active Reminders

Please follow the following steps:

  • This extension let's you set a webpage to open at a particular time whenever you wish to come back to it. The way it works is, when the reminder reaches the time you set, the extension will open a new window with the webpage you have set to open.

  • You can add a new reminder using the plus button in the popup page, a new page with a form to fill up will appear, where you will fill up all the details to add a new reminder.

  • The popup page will contain all your active reminders, it also includes link to your completed reminder page which will contain all the finished reminders.

Click the Scrinota icon to return to popup page
